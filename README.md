# YouTube MP3 and MP4 Converter

A free and open-source tool to convert and download YouTube videos to MP3 or MP4 format.

## Description

This project allows you to convert any YouTube URL to either MP3 (audio) or MP4 (video) format and download them directly to your device. It's designed to be simple, efficient, and completely free to use.

## Features

- 🎵 Convert YouTube videos to MP3 (audio only)
- 🎬 Download YouTube videos as MP4 (video with audio)
- 🆓 Completely free to use
- 🚀 Simple and easy to use interface
- ⚡ Fast conversion and download
